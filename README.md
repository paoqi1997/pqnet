pqnet是一个面向Linux的、基于Reactor模式的网络库，由C++编写。

## Install

通过 make 安装 pqnet。

```
$ make
$ sudo make install
```

通过 CMake 安装 pqnet。

```
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
$ sudo make install
```
